# 🚀 Simo's Dotfiles

My personal Linux development environment.

This repository contains the configuration files, tools, and scripts I use every day for C/C++, Python, Docker, Linux development, cybersecurity, and Neovim.

The goal is simple:

> Clone one repository, run one script, and have a complete development environment in minutes.

---

# ✨ Features

* ⚡ Neovim IDE
* 🐚 Zsh + Oh My Zsh
* ⭐ Starship prompt

* 🪟 tmux configuration
* 📦 Automatic installer

# 🚀 Installation

Clone the repository

```bash
git clone --recursive git@github.com:MohamedHaouari0/dotfiles.git
```

Enter the repository

```bash
cd dotfiles
```

Run the installer

```bash
chmod +x install.sh
./install.sh
```
